## Installation

- [Sign up (free) on comet.ml and obtain an API key](https://www.comet.ml)

- **Install Comet.ml from PyPI:**

```sh
pip install comet_ml
```
Comet.ml python SDK is compatible with: __Python 2.7-3.6__.
